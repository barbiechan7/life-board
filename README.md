# Life Board

やること・夢・目標・メモ・やりたいことを管理する個人用Webサイトです。

## 公開URL

https://barbiechan7.github.io/life-board/

## ローカルで開く

```bash
cd life-board-site
python3 -m http.server 8080
```

http://localhost:8080 を開いてください。

## 機能

- やること / 夢 / 目標 / メモ / やりたいこと
- 検索・フィルタ・ピン留め
- データはブラウザに自動保存
- JSON エクスポート / インポート
